# Monopoly India

An online multiplayer Monopoly game featuring Indian cities. Currently in development.

https://monopoly-client-six.vercel.app/

## Features
- **Multiplayer**: Real-time gameplay using Socket.io.
- **Indian Theme**: Properties are based on Indian cities.
- **Game Logic**: Buy properties, pay rent, and roll dice.

## Tech Stack
- **Frontend**: React, Vite
- **Backend**: Node.js, Express, Socket.io
- **Styling**: CSS

## How to Run Locally

### Prerequisites
- Node.js installed.

### Steps
1. **Server**:
   ```bash
   cd server
   npm install
   npm start
   ```

2. **Client**:
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.
